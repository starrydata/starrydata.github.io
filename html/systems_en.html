Web Systems

Starrydata2 web system

www.starrydata2.org

A large-scale open database that collects and structures over 200,000 curves of material property data extracted from graph images in published papers.

Visit System ↗

Starrydata Visualizer

visualizer.starrydata.org

Provides visualized graphs to help understand the overview of big data registered in Starrydata2 web system. Data is updated daily.

Visit System ↗

Starrydata Sample Explorer

explorer.starrydata.org

Allows you to explore data stored in Starrydata2 web system based on chemical composition. By applying composition filters, you can extract samples that meet specified criteria. You can also access plots showing clustering results and download data files.

Visit System ↗

Starrydata Explorer 3D

explorer-3d.starrydata.org

Allows you to view the distribution of thermoelectric properties of samples registered in Starrydata2 web system as 3D plots. These graphs show the distribution of published samples against Seebeck coefficient [V/K], log10[S/m], and lattice thermal conductivity [W/mK].
